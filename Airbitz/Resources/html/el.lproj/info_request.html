Ζητήστε Bitcoin

Στην συγκεκριμένη οθόνη μπορείτε να ζητήσετε bitcoin από κάποιον άλλο χρήστη. Επιλέξτε το πορτοφόλι που προορίζεται να δεχθεί τα χρήματα κάνοντας κλικ στο όνομα του πορτοφολιού στην κορυφή.

Δείξτε τον QR κωδικό στον Αποστολέα ώστε να τον σαρώσει και να σας στείλει bitcoin. Μόλις γίνει λήψη των bitcoin, η συγκεκριμένη οθόνη θα εντοπίσει αυτόματα την συναλλαγή και θα προχωρήσει στις Λεπτομέρειες Συναλλαγής όπου μπορείτε να παρακολουθήσετε τη συναλλαγή σας.

Πατήστε τα κουμπία [Email] ή [SMS] για να στείλετε αίτημα σε κάποιον απομακρυσμένο χρήστη μέσω email ή SMS.

Μπορείτε επίσης να αντιγράψετε τη διεύθυνση αυτής της αίτησης χρησιμοποιώντας το κουμπί [Copy] και να κάνετε επικόλληση έπειτα σε κάποια άλλη εφαρμογή και πορτοφόλι για να στείλετε χρήματα.

Μπορείτε προαιρετικά να εισάγετε ένα ποσό στην αίτηση πατώντας στο πεδίο [Εισάγετε Ποσό] ή πατώντας στον κωδικό QR. Πατήστε το κουμπί δεξιά στην κορυφή για να επιλέξετε μεταξύ bitcoin ή fiat (USD, EUR). Εάν δεν εισαχθεί κανένα ποσό, ο Αποστολέας θα καθορίσει την ποσότητα bitcoin που θα στείλει.
