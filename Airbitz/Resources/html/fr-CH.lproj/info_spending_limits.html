Limites de dépenses

[Limite de Dépenses Journalière] limitera votre compte à dépenser le montant défini de bitcoin par jour. Le mot de passe du compte sera exigé pour dépenser au-delà de ce montant.

[Exiger PIN au delà montant] exigera l’utilisateur à entrer son code PIN lors de l’envoi de fonds au-delà de la quantité spécifiée. La désactivation de ce paramètre annulera toute exigence de PIN pour des dépenses.

Prenez note qu’un code PIN n’est pas nécessaire dans les deux premières minutes suivant la connexion à votre compte.
