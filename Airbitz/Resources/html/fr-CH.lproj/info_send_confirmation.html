Envoyer la Confirmation

Entrez la valeur à envoyer dans le Bitcoin ou fiat (c’est à dire. USD/EUR) champs de monnaie. La valeur de la devise secondaire sera automatiquement calculée à base du taux d'échange actuel.

Appuyez sur [Max] pour envoyer le maximum d’argent de votre porte-monnaie actuellement sélectionné. Notez que le montant maximum peut être inférieur au solde actuel en raison des frais de transaction ou de transactions antérieures non confirmées.

Tapez sur la dénomination du bitcoin [bits] pour modifier la taxe minière recommandé enfin d’envoyer avec cette transaction
Tapez le code de devise fiat, c’est à dire. « USD », pour changer la devise fiat utilisée pour convertir vers et à partir de Bitcoin.
Si nécessaire, entrez votre retrait PIN ou mot de passe et faites glisser le bouton vers la gauche pour envoyer votre transaction.

Notez que le solde total de votre porte-monnaie peut ne pas être disponible pour le retrait vu que des transactions antérieures pourront être à confirmer avant que les fonds qui leur sont associés pourront être utilisés. Chaque transaction exige également une petite taxe de transaction ce qui réduit le montant total qui peut être envoyé du portefeuille.
