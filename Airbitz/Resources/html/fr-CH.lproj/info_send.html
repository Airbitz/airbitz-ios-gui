Scanner et Envoyer Bitcoin

Cet écran vous permet de scanner un code QR afin d'envoyer du bitcoin ou d'importer (balayer) une clé dans votre portefeuille.

Vous pouvez scanner un code QR BitID pour vous connecter à un site Web à l’aide de BitID.

Vous pouvez aussi Edge Login dans une Application décentralisée comme Augur en scannant le code QR.

Choisir un portefeuille de votre compte en cliquant sur le nom du portefeuille sur l’en-tête en haut de la page.

Scanner le code QR d'un autre portefeuille pour obtenir une adresse du portefeuille pour envoyer du montant pour.

Appuyez sur le bouton [Photos] pour choisir une photo contenant un code QR que vous souhaitez numériser.

Tapez [Address] pour manuellement tapez ou collez une adresse donc vous voulez envoyer de l’argent.

S’il y a des demandes de paiement bluetooth près de vous, elles apparaîtront en bas. Tapez sur une demande pour envoyre à ce destinataire.

Tapez [Transfert] pour transférer bitcoin à un autre portefeuille dans votre compte. Simplement taper le nom de destination du portefeuille dans le menu contextuel.
