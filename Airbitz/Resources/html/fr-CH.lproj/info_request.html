Demander Bitcoin

Cet écran vous permet de demander du bitcoin à un autre utilisateur. Choisissez le porte-monnaie de destination en cliquant sur le nom du porte-monnaie dans l’en-tête du haut de la page.

Présenter le code QR à l’expéditeur et demandez-leur de scanner le code pour vous envoyer des bitcoin. Une fois que le bitcoin est reçu, cet écran va automatiquement détecter la transaction et procéder à tous les détails de Transaction où vous pouvez marquer votre transaction.

Appuyez sur les boutons [Email] ou [SMS] pour envoyer une demande à un utilisateur distant par e-mail ou SMS.

Vous pouvez également copier l’adresse de cette demande à l’aide de la touche [Copier] et collez-le dans une différentes application de portefeuille pour envoyer des fonds.

Vous pouvez éventuellement entrer un montant à demander en tapant dans le champ de texte [Entrez le montant] ou en tapant le code QR. Appuyez sur le bouton droit pour choisir entre bitcoin ou fiat (USD, EUR). Si aucun montant n’est entré, l’expéditeur déterminera le montant du bitcoin à envoyer.
