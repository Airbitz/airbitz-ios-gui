Escaneie e Envie Bitcoin

Esta tela permite a você Escanear um código QR para enviar Bitcoin ou importar (buscar) uma chave privada para sua carteira.

Você pode digitalizar um código BitID QR para fazer login em um site usando BitID.

Você também pode efetuar Edge login para outro aplicativo descentralizado, como o Augur, escaneando o código QR.

Escolha uma carteira da sua conta clicando no nome da carteira no cabeçalho superior.

Escaneie o código QR de outra carteira para obter o endereço da carteira para envio de fundos.

Toque em [Photos] Para escolher uma foto contendo um código QR que você gostaria de escanear.

Toque em [Address] para digitar manualmente ou colar um endereço para enviar dinheiro.

Se houver solicitações de pagamento bluetooth perto de você, elas aparecerão na parte inferior. Toque em uma solicitação para enviar para esse destinatário.

Toque em [Transfer] para transferir bitcoin para outra carteira em sua conta. Basta tocar no nome da carteira de destino no pop-up.
