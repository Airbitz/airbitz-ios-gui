Informações para Exportação de Carteira


Escolha uma carteira para exportar usando o botão no cabeçalho.

Selecione [CSV] para exportar uma lista CSV de todas as transações para a sua carteira selecionada.

Selecione [Quickbooks] para exportar um arquivo no formato QBO para importar em Quickbooks.

Selecione [Wallet Private Seed] para exportar o seed privado da carteira selecionada.
Selecione [Wallet Public Seed] para exportar o seed público (XPub) da carteira selecionada.
AVISO, o seed privado dá acesso a todos os fundos na carteira selecionada. Mantenha o fundo de capital seguro e não o exponha aos outros.
