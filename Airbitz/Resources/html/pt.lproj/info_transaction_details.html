Detalhes de Transação

Toque no campo superior (Beneficiário ou Nome da Empresa) para digitar o nome da pessoa ou negócio que você está transacionando. Este campo irá se autocompletar com empresas que estão nas proximidades do seu catálogo de endereços. Você sempre poderá alterar este campo.

Toque no botão de Categoria para escolher uma categoria principal para esta transação. As categorias são Rendas, Despesas, Transferências e Trocas. Toque no campo de texto da categoria para escolher uma subcategoria como "Alimentação e Jantar". Digite em uma nova categoria para adicioná-la à lista.
