Bitcoins scannen und senden

Auf diesem Bildschirm können Sie QR-Codes scannen um Bitcoin zu senden oder Ihrer Wallet einen privaten Schlüssel zukommen zu lassen.
Sie können auch einen BitID QR-Code bei der Anmeldung bei auf Website mit BitID scannen.
Wählen Sie eine Wallet aus Ihrem Konto durch Klicken auf den Wallet-Namen in der oberen Kopfzeile aus.

Scannen Sie den QR-Code von einer anderen Wallet, um die Adresse der Wallet abzurufen, an die Sie Gelder überweisen möchten.

Tippen Sie auf die Schaltfläche "[Fotos/Photos]" und wählen Sie ein Foto mit einem QR-Code aus, den Sie scannen möchten.

Tippen Sie auf [Adresse/Address] um manuell eine Adresse, an die Sie Geld senden möchten, einzugeben oder einzufügen.

Wenn Bluetooth-Zahlungsanordnungen in Ihrer Nähe sind, werden Sie unten auf der Seite angezeigt. Tippen Sie auf eine Anfrage um an diesen Empfänger zu senden.

Tippen Sie auf [Transfer] um Bitcoin an ein weiteres Wallet in Ihrem Konto zu übertragen. Tippen Sie einfach den Zielnamen der Wallet im Popup an.
