Bitcoin scannen und senden

Auf diesem Bildschirm können Sie QR-Codes scannen um Bitcoin zu schicken oder einen privaten Key in ihr Wallet zu importieren.

Sie können einen BitID QR-Code scannen, um sich auf eine Webseite anzumeldung die BitID verwendet.

Ebenfalls können Sie sich per Edge-Anmeldung bei anderen dezentralen Apps, wie z. B. Augur anmelden, indem Sie den QR-Code scannen.

Wählen Sie eine Wallet aus Ihrem Konto, indem Sie auf den Namen der Wallet in der Kopfzeile klicken.

Scannen Sie den QR-Code einer Wallet um die Adresse für Überweisungen der Wallet zu erhalten.

Klicken Sie auf die [Photos] Taste, um aus den Fotos mit QR-Codes zu wählen.

Tippen Sie auf [Address] um manuell oder per Einfügen eine Adresse einzugeben und einen Geldbetrag zu überweisen.

Falls es Zahlungsaufforderungen via Bluetooth in Ihrem näheren Umfeld gibt, werden Ihnen diese auf der unteren Seite angezeigt. Tippen Sie auf die Anforderung, um diese an den Empfänger zu senden.

Tippen Sie auf [Transfer] um Bitcoin in einem anderen Wallet auf Ihr Konto zu überweisen. Klicken Sie einfach den Namen der Ziel-Wallet im Popup-Fenster an.
