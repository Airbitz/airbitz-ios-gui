Details der Transaktion

Tippen Sie auf das obere Feld (Zahlungsempfänger oder Firmenname), geben Sie den Namen der Person oder des Unternehmens ein, mit denen Sie ein Geschäft tätigen. Dieses Feld wird von Geschäften in der Nähe automatisch vervollständigt und aus Ihrem Adressbuch. Sie können dieses Feld immer zu einem späteren Zeitpunkt ändern.

Tippen Sie auf die Schaltfläche "Kategorie" wählen Sie eine Hauptkategorie für diese Transaktion wählen. Kategorien sind Einkommen, Kosten, Transfers oder Austausch. Tippen Sie auf das Kategorie-Text-Feld, um eine Unterkategorie wie "Lebensmittel und Speisen" zu wählen. Geben Sie in eine neue Kategorie ein, um der Liste eine Kategorie hinzuzufügen.
