Solicitud de Bitcoin

Esta pantalla le permite solicitar bitcoin de otro usuario. Elegir la carpeta de destino haciendo clic en el nombre de la carpeta en la cabecera superior.

Presente código QR al remitente y pídale que escanear el código para enviar bitcoin. Una vez recibida la bitcoin, esta pantalla automáticamente detectará la transacción y proceder a los detalles de la transacción donde usted puede etiquetar su transacción.

Pulse los botones [Email] o [SMS] para enviar una solicitud a un usuario remoto a través de correo electrónico o SMS.

También puede copiar la dirección de esta solicitud con el botón [Copy] y pégalo en una aplicación de carpeta diferente para enviar fondos.

Opcionalmente puede ingresar una cantidad a solicitar pulsando en el campo de texto [Escriba cantidad] o pulsando el código QR. Pulse el botón superior derecho a elegir entre bitcoin o fiat (USD, EUR). Si no se especifica ninguna cantidad, el remitente determinará la cantidad de bitcoin para enviar.
