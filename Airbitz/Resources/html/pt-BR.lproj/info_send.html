Escaneie e Envie Bitcoin

Esta tela permite a você Escanear um código QR para enviar Bitcoin ou importar (pesquisar) uma chave privada para sua carteira.

Você pode digitalizar um código BitID QR para fazer login em um site usando BitID.

Você também pode acessar o Edge para outro aplicativo descentralizado, como o Augur, digitalizando o código QR.

Escolha uma carteira da sua conta clicando no nome da carteira no cabeçalho superior.

Digitalizar o código QR de outra carteira para obter o endereço da carteira para enviar fundos para.

Toque em [Photos] Para escolher uma foto contendo um código QR que você gostaria de digitalizar.

Toque [Address] Para digitar manualmente ou colar um endereço para enviar dinheiro para.

Se houver solicitações de pagamento bluetooth perto de você, elas aparecerão na parte inferior. Toque em um pedido para enviar para esse destinatário.

Toque [Transfer] Para transferir bitcoin para outra carteira em sua conta. Basta tocar no nome da carteira de destino no pop-up.
