Informações para Exportar para a Carteira


Escolha uma carteira para exportar usando o botão no cabeçalho.

Selecione [CSV] para exportar uma lista em formato CSV de todas as transações para a carteira selecionada.

Selecione [Quickbooks] para exportar um arquivo em formato QBO para o Quickbooks.

Selecione [Wallet Private Seed] para exportar os fundos de capital privado da carteira selecionada.
ATENÇÃO, os fundos de capital privado dão acesso a todos os fundos na carteira selecionada. Mantenha o fundo de capital seguro e não o exponha aos outros.
