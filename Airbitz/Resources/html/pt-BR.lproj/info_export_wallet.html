Informações para Exportar para a Carteira


Escolha uma carteira para exportar usando o botão no cabeçalho.

Select [CSV] to export a CSV list of all transactions for your selected wallet.

Select [Quickbooks] to export a QBO format file for import into Quickbooks.

Select [Wallet Private Seed] to export the private seed of the selected wallet.
Select [Wallet Public Seed] to export the public seed (XPub) of the selected wallet.
WARNING, the private seed gives access to all funds on the selected wallet. Mantenha o fundo de capital seguro e não o exponha aos outros.
