Solicitar Bitcoin

Esta tela permite que você solicite bitcoin de outro usuário. Escolha a carteira de destino clicando no nome dela no cabeçalho superior.

Apresente o código QR ao Remetente, que irá escaneá-lo para lhe enviar bitcoin. Uma vez recebido o bitcoin, esta tela irá automaticamente detectar a transação e prosseguir para os Detalhes da Transação onde você poderá marcar sua transação.

Toque no botão [Email] ou [SMS] para enviar uma solicitação para um usuário remoto via e-mail ou SMS.

Você também pode copiar o endereço dessa solicitação usando o botão [Copy] e colá-lo em uma aplicação de carteira diferente para enviar fundos.

Opcionalmente, você pode digitar uma quantia para solicitar clicando no campo de texto [Enter Amount] ou clicando no código QR. Toque no botão superior direito para escolher entre bitcoin ou fiat (USD, EUR). Se nenhuma quantia for inserida, o Remetente irá determinar a quantia de Bitcoin para enviar.
