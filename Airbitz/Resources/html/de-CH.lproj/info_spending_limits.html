Ausgabelimits

[Tägliches Ausgabenlimit] wird Ihr Konto auf die festgelegte Menge von Bitcoins pro Tag beschränken. Das Passwort wird benötigt, wenn sie über diese Menge hinaus ausgeben möchten.

[Erfordere PIN über Betrag] wird den Benutzer dazu auffordern, seine PIN einzugeben, sollte er Guthaben über dem angegebenen Betrag senden. Durch Deaktivierung dieser Einstellung wird bei keinen Ausgabe eine PIN benötigt.

Beachten Sie, dass in den ersten zwei Minuten nach dem Einloggen keine PIN benötigt wird.
