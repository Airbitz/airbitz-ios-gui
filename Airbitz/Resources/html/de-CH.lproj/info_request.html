Bitcoin anfordern

Dieser Bildschirm erlaubt Ihnen Bitcoins von einem anderen Benutzer anzufordern. Wählen Sie die Ziel-Wallet durch einen Klick auf den Namen der Wallet in der Kopfzeile oben.

Zeigen Sie den QR Code dem Absender und lassen Sie ihn den Code scannen um Ihnen Bitcoin zu senden. Sobald Bitcoin empfangen wurde, wird der Bildschirm die Transaktion automatisch erkennen und zu den Transaktionsdetails fortfahren, wo Sie Ihre Transaktion markieren können.

Tippen Sie auf die [Email] oder [SMS] Schaltflächen, um eine Anforderung an ein Remote-Benutzer per E-Mail oder SMS zu senden.

Sie können auch die Adresse dieser Anforderung mithilfe der Schaltfläche "[Copy]" kopieren und dann in eine andere Wallet-Anwendung einfügen, um Geld zu überweisen.

Sie können optional einen Betrag eingeben, indem Sie in das Textfeld [Betrag eingeben] tippen oder durch Antippen des QR-Codes. Tippen Sie rechts oben auf die Schaltfläche um zwischen Bitcoin und Fiat (USD, EUR) umzuschalten. Wenn kein Betrag eingegeben wird, bestimmt der Sender die Höhe der zu sendenden Bitcoin.
