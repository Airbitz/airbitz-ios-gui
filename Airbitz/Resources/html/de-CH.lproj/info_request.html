Bitcoin anfordern

Dieser Bildschirm erlaubt Ihnen Bitcoins von einem anderen Benutzer anzufordern. Wählen Sie das Ziel-Wallet durch einen Klick auf den Namen der Wallet in der Kopfzeile oben.

Zeigen Sie den QR Code dem Sender und lassen Sie ihn die Zahlung scannen um Ihnen Bitcoins zu senden. Sobald Bitcoin eingegangen ist, wird dieser Bildschirm automatisch die Transaktion erkennen und mit den Transaktionsdaten fortfahren, wo Sie Ihre Transaktion markieren können.

Tippen Sie auf die [Email] oder [SMS] Schaltflächen, um eine Anforderung an ein Remote-Benutzer per E-Mail oder SMS zu senden.

Sie können auch die Adresse dieser Anforderung mithilfe der Schaltfläche "[Copy]" kopieren und dann in eine andere Wallet-Anwendung einfügen, um Geld zu überweisen.

Optional können Sie eine bestimmte Menge auffordern, indem Sie im [Betrag eingeben/Enter Amount] Text-Feld tippen oder durch Tippen auf den QR-Code. Tippen Sie oben recht auf die Umschaltfläche, um zwischen Bitcoin oder Fiat (USD, EUR) zu wählen. Wenn kein Betrag eingegeben wird, wird der Absender die Höhe der Bitcoin bestimmen.
