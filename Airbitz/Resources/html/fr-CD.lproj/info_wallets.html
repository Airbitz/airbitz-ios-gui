Liste de Porte-Monnaies

Créer un nouveau portefeuille utilisant le signe [+] le bouton sur le PORTEFEUILLES en-tête. Choisissez une monnaie fiat que vous aimeriez utiliser pour suivre la valeur des transactions dans le nouveau portefeuille. Notez que ce paramètre monétaire de fiat n’est pas modifiable une fois réglé. Vous pouvez modifier la devise fiat global pour votre compte dans l’onglet Paramètres dans le coin inférieur droit.

Renommez ou supprimez un portefeuille par tap et contenant un nom de portefeuille pendant 3 secondes

Réorganiser les portefeuilles en faisant glisser la barre de 3 icône à droite de chaque portefeuille. Le portefeuille le plus haut est votre portefeuille par défaut utilisée pour demander et envoyer des transactions quand d’abord connecté

Portefeuilles d'archive en les traînant ci-dessous le ARCHIVES en-tête. Les portefeuilles archivés n’apparaissent pas dans la balance en haut de la barre et ne peuvent pas être utilisés pour envoyer ou demander des fonds. Les archives peuvent être masqués en cliquant sur la flèche vers la droite sur la ARCHIVES en-tête.
